# 二维数组的一维表示

```java
int[][] nums = new int[r][c];

// 二维映射到一维
int[] arr = new int[r*c];
for(int i = 0; i < r; i++) {
    for (int j = 0; j < c; j++) {
        arr[i*c+j] = nums[i][j];
    }
}

// 一维映射到二维
for(int i = 0; i < r*c; i++) {
    nums[i / c][i % c] = arr[i];     // c 为列 
}
```

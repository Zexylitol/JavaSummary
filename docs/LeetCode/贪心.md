# LeetCode 45 跳跃游戏II

- 双指针 + 贪心 + dp
  - 时间复杂度：每个位置最多被扫描两次（同时被$i$和$j$经过），复杂度为$O(n)$

```java
class Solution {
    public int jump(int[] nums) {
        int n = nums.length;
        int[] f = new int[n]; 
        for (int i = 1, j = 0; i < n; i++) {
            while (j + nums[j] < i) j++;
            f[i] = f[j] + 1;
        }
        return f[n - 1];
    }
}
```
